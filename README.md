# Rugby Manager

**CLI-Based Rugby Management Simulation Game**  

**Language:** Python  
**Technologies:** Command-line interface, Object-Oriented Programming, Game Logic Simulation  

---

## Description

Rugby Manager is a command-line sports management simulation game where you take control of a rugby team. Manage players, plan strategies, simulate matches, and compete to lead your team to victory.  

This project demonstrates:  

- Designing a game simulation entirely in Python  
- Implementing object-oriented programming for teams, players, and matches  
- Managing complex logic and game rules in a structured way  
- Handling user input and command-line interaction

## Features

- Create and manage multiple teams  
- Track player stats and performance  
- Simulate matches with realistic scoring and outcomes  
- League standings   

---

## Usage

Run the game with:
python main.py

### Follow the on-screen prompts to:
- Select teams
- Manage players
- Simulate matches
- View standings
